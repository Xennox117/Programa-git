ETIQUETAS
En este caso se seleccionan las etiquetas principales del HTML para darles estilos y mejora al sitio web. Como lo son:

body
Se utiliza para aplicar estilos globales al contenido del cuerpo del documento.

header
Se utiliza para aplicar estilos al encabezado del documento.

nav
Se utiliza para aplicar estilos a la sección de navegación.

nav ol
Selecciona las listas ordenadas ol dentro del elemento nav y se utiliza para aplicar estilos específicos a esas listas.

nav ol li
Selecciona los elementos de lista li dentro de las listas ordenadas ol en el elemento nav.

nav ol li a
Selecciona los enlaces a dentro de los elementos de lista li en las listas ordenadas ol dentro del elemento nav.

nav ol li:hover .submenu
Selecciona el submenú .submenu dentro del elemento li que se está pasando por encima.

nav .submenu
Selecciona los elementos con la clase .submenu dentro del elemento nav.

nav .submenu li
Selecciona los elementos de lista li dentro del submenú .submenu.

nav .submenu li a
Selecciona los enlaces a dentro de los elementos de lista li en el submenú .submenu.

nav .submenu li a:hover
Selecciona los enlaces a dentro de los elementos de lista li en el submenú .submenu cuando se pasa el ratón por encima.

main
Se utiliza para aplicar estilos al contenido principal del documento.

section
Se utiliza para aplicar estilos a las secciones del documento.

h2
se utiliza para aplicar estilos a estos encabezados.

footer
Se utiliza para aplicar estilos al pie de página del documento.
