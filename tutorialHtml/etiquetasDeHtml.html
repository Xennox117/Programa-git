Etiquetas de la imagen de ejemplo HTML

El codigo de la imagen de ejemplo es el codigo utilizado para la creacion de la pagina index. Estas definicones van de manera descendente conforme al codigo


DOCTYPE
Declara el tipo de documento y la versión de HTML que se está utilizando. En este caso, HTML.

html lang="es"
Etiqueta raíz que envuelve todo el contenido HTML. El atributo lang especifica el idioma del contenido, en este caso, español.

head
Contiene meta-información sobre el documento, como el título, enlaces a hojas de estilo, y otros metadatos.

meta charset="UTF-8"
Especifica la codificación de caracteres del documento, en este caso, UTF-8, lo que permite usar la mayoría de los caracteres de los idiomas del mundo.


meta name="viewport" content="width=device-width, initial-scale=1.0"
Proporciona información al navegador sobre cómo controlar las dimensiones y la escala de la página, esencial para el diseño responsivo.

title
Define el título del documento que aparece en la pestaña del navegador.

link rel="stylesheet" href="css/new.css"
Vincula una hoja de estilo CSS externa al documento HTML. El css es la carpeta de destino y el new.css es el archivo.

body
Contiene el contenido principal del documento HTML, visible para los usuarios.

header
Sección de encabezado que generalmente contiene información introductoria o de navegación.

h1
Define un encabezado de primer nivel, usado para títulos principales.

nav
Contiene enlaces de navegación.

ol
Define una lista ordenada.

li
Define un elemento de una lista.

a href="#html"
Define un enlace, en este caso, un enlace ancla que lleva a una sección específica del mismo documento. En este caso lleva al destino CSS o HTML.

main
Contiene el contenido principal del documento.

section
Define una sección del documento, usada para agrupar contenido relacionado.

h2
Define un encabezado de segundo nivel, usado para subtítulos.

img src="img/html.png" alt="Ejemplo de HTML"
Inserta una imagen en el documento. El atributo src define la ruta de la imagen, y alt proporciona un texto alternativo. Donce img representa la carpeta de destino y html.png la imagen.

p
Define un párrafo de texto.

footer
Contiene información de pie de página, como derechos de autor y enlaces adicionales.
